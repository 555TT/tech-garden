# 工具与资源

## 栏目定位

工欲善其事，必先利其器。本栏目是我日常开发中使用的高效工具、实用资源和技巧的集散地，扮演博客的“弹药库”角色。

**一句话描述：** 那些让开发效率翻倍的利器，和值得反复翻阅的宝藏资源。

## 内容范围

| 子领域       | 示例主题                                                     |
| ------------ | ------------------------------------------------------------ |
| 开发工具     | IDE 插件推荐与配置、终端美化、Git 高级技巧、数据库客户端评测 |
| 命令行艺术   | awk/sed/jq 实战、Shell 脚本复用、一行命令解决问题            |
| 效率工具     | Alfred/Raycast 工作流、跨设备剪贴板、窗口管理                |
| 常用库与框架 | Go 语言实用三方库推荐、通用工具类库整理                      |
| 资源推荐     | 后端必读书单、优质博客与 Newsletter、值得关注的 GitHub 仓库  |

## 写作原则

- **精而不滥：** 只推荐自己长期使用并验证过的工具，拒绝“工具收藏癖”。
- **场景明确：** 每个工具必须说明适用场景和解决的问题，不做功能列表翻译。
- **可操作：** 尽量提供配置文件和开箱即用的方案。

## 与其他栏目的边界

- 若文章重点是“GitHub Actions 配置”，归 **云原生与部署运维**。
- 若文章重点是“Copilot 和 Cursor 的比较”，归 **AI Coding 时代**。
- 本栏目侧重工具本身的推荐、配置与使用技巧，与技术架构和语言深度无关。

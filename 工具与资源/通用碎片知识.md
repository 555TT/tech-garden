### 📖 JSONPATH表达式常用语法速查表

| 语法     | 说明                     | 示例                        |
| :------- | :----------------------- | :-------------------------- |
| $        | 根节点（入口）           | $                           |
| .        | 子节点访问               | $.store.name                |
| ..       | 递归下降（任意深度）     | $..price                    |
| *        | 通配符（所有子元素）     | $.store.*                   |
| [n]      | 数组下标（从 0 开始）    | $.store.book[0]             |
| [-1]     | 最后一个元素             | $.store.book[-1]            |
| [m:n]    | 数组切片                 | $.store.book[0:2]           |
| [*]      | 数组所有元素             | $.store.book[*]             |
| [?(...)] | 过滤器表达式             | $.store.book[?(@.price<10)] |
| @        | 当前节点（过滤器内使用） | @.price                     |
| ['key']  | 带特殊字符的键名         | $['my-key']                 |

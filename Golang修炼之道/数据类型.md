对于golang的内置数据类型，占用内存大小如下：

| 类型                          | 字节数           | 备注                                                         |
| ----------------------------- | ---------------- | ------------------------------------------------------------ |
| bool                          | 1个字节          | 1                                                            |
| intN, uintN, floatN, complexN | N/8 个字节       | N/8                                                          |
| int, uint, uintptr            | 计算机字长/8     | 8                                                            |
| *T, map, func, chan           | 计算机字长/8     | 8                                                            |
| string                        | 2 * 计算机字长/8 | 16。data、len分别8个字节                                     |
| interface                     | 2 * 计算机字长/8 | 16。tab（非空接口时）、data或_type(空接口时)、data分别8个字节 |
| []T                           | 3 * 计算机字长/8 | 24。array、len、cap分别8个字节                               |

计算机字长就是 CPU 的原生数据处理位数，也就是我们常说的"32位"或"64位"。

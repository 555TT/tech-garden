# 后端工程师的成长手记

> 以 Go 语言为剑，以云原生与架构为盾，左手 AI Coding，右手技术人文。

## 📖 关于本博客

这是一个后端工程师的个人技术博客仓库，记录我在技术道路上的实践总结与思考提炼。

**我的技术背景：** 大学毕业前一直学习 Java 开发，现专注 Go 生态。这里的内容不是教程搬运，而是来自真实项目中的踩坑、复盘和生产验证后的经验。

**核心方向：** 编程思想、架构设计、问题解决方法论、软技能与职业成长。它们是不会随语言更替而失效的长期价值。
